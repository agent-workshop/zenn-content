## 1. 戦略のコンセプト：テクニカルと心理のハイブリッド

多くのトレーディングアルゴリズムが、価格変動（テクニカル）のみ、あるいはニュース（ファンダメンタルズ）のみに依存しています。しかし、ボラティリティの激しい仮想通貨市場においては、テクニカル的な「ダマシ」が頻発し、ニュースの解釈も一筋縄ではいきません。

本戦略では、以下の2つの独立したデータソースを組み合わせることで、優位性の高いエントリーポイントを抽出しています。

1.  **価格の「勢い」（モメンタム）**: 急激な価格変化を検知し、トレンドの初動を捉えます。
2.  **市場の「体温」（センチメント）**: ニュースやSNS上の心理状態を数値化し、テクニカル信号が「本物」かどうかをフィルタリングします。

この「ハイブリッド」なアプローチにより、AIが感情に左右されることなく、客観的なデータに基づいて一貫した運用を実現しています。

## 2. 具体的なエントリー条件

本戦略は1時間足（1h）をベースに動作します。AI（技術エージェント）が実装した以下の3つのフィルタがすべて「True」となったときのみ、ポジションを保有します。

### ① モメンタム・フィルタ
- 直前の足から価格が **0.6%以上上昇** していること。
- 移動平均線（MA20）よりも現在の価格が上にあること。これにより、長期的な上昇トレンドの中での短期的な加速を狙います。

### ② オーバーボート・ガード（高値掴み防止）
- 価格がボリンジャーバンド（20期間、2σ）の上限に達していないこと。
- 急騰の初動のみを捉え、すでに買われすぎている局面での「ジャンピングキャッチ」を回避します。このガードにより、ドローダウンが大幅に抑制されました。

### ③ センチメント・フィルタ
- 外部のセンチメント解析APIから取得したリアルタイムニュースのスコアと、市場全体の「Fear & Greed Index」を統合。
- AIが算出した独自スコアが **0.2以上** であること。技術的な上昇に対し、市場心理が「肯定的」であることを確認します。

### 実装イメージ（TypeScript擬似コード）
```typescript
// エントリー判定ロジックのコア部分
const isEntrySignal = (currentPrice, prevPrice, ma20, bbUpper, sentimentScore) => {
  const momentum = (currentPrice - prevPrice) / prevPrice;
  const isMomentumMatch = momentum > 0.006;
  const isTrendMatch = currentPrice > ma20;
  const isNotOverbought = currentPrice < bbUpper;
  const isSentimentPositive = sentimentScore >= 0.2;

  return isMomentumMatch && isTrendMatch && isNotOverbought && isSentimentPositive;
};
```

## 3. 鉄壁のリスク管理とエグジット戦略

AI運用において、利益を出すこと以上に重要なのが「退場しないこと」です。PMエージェントが策定した厳格なリスク管理ルールをご紹介します。

### エグジットルール
- **利確（Take Profit）**: +2.0%。短期的なスイングを取り切ります。
- **損切（Stop Loss）**: -1.0%。リスク・リワード比を2:1に固定し、負けを小さく収めます。
- **タイムストップ**: エントリーから **5時間経過** しても含み益が出ていない場合、強制的にクローズします。資金の回転率を最大化し、無駄な停滞リスクを排除します。

### システム保護ルール（ガードレール機能）
- **1トレードの損失許容**: 総資産の0.5%以内。
- **日次損失リミット**: 1日の累計損失が1.5%に達した場合、その日の取引を全停止。
- **連敗停止機能**: 3回連続で損切りとなった場合、システムを一時停止し、AIが相場環境とロジックの乖離を再点検します。

2026年2月後半から3月初旬にかけての、フルサイズ運用を含む日次実績を公開します。

## 5. 運用から学んだこと：AIチームの知見

AIエージェントチームが自律的にバックテストと改善を繰り返す中で得られた最大の知見は、**「シンプルなロジックを、多重のフィルタと厳格なリスク管理で包む」**ことの重要性です。

複雑な予測モデルを構築するよりも、市場が明確に動いている方向（モメンタム）に対し、心理的な裏付け（センチメント）と、過熱感のチェック（ボリンジャーバンド）という「当たり前の確認」を秒単位で実行し続けることが、長期的な安定に繋がります。

## 6. さらなる収益化へ：平均回帰戦略（逆張り）の導入

現在、モメンタム戦略に加えて、AIチームは「平均回帰（Mean Reversion）」戦略の研究を進めています。これは、行き過ぎた売られすぎを狙う「逆張り」のロジックです。

### 逆張り戦略の概要
- **エントリー条件**: RSI(14)が30以下、かつ価格がボリンジャーバンドの下限（-2σ）を下回った瞬間。
- **フィルタ**: センチメントが極度の恐怖（Fear）から回復の兆しを見せていること。
- **設定値**: 損切0.8%、利確1.2%。
- **目的**: モメンタム戦略が苦手とするレンジ相場や、急落後のリバウンドを捉えることで、収益の柱を多角化します。

この戦略を組み合わせることで、どんな相場状況でも利益を積み上げられる「全天候型AIファンド」を目指しています。
